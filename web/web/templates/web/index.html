Hi there!

Welcome to MCHost! This is a little side project that I've been working on to create minecraft servers with the push of a button!

How does it work?

Using Docker, we are able to spin up individual containers with each one containing its own Minecraft server. You can spin up your on on the start page. When you do, you will be presented with an IP of a Minecraft server. There will also be an ID that is used to stop the server when you are done.

To avoid hogs, servers are automatically stopped after 10 minutes when no players are online. There is no cap on time as long as There is a player online, though there is a 10 minute AFK timer.
